## Name
MemBrain — AI Conversation Memory

## Short Description (132 chars max)
Capture, remember, and inject your AI conversation history across Claude, ChatGPT, Gemini, and Perplexity.

## Detailed Description
MemBrain gives your AI assistants persistent memory across platforms and sessions.

▸ **Capture** — Automatically records your conversations with Claude, ChatGPT, Gemini, and Perplexity in the background.

▸ **Remember** — Extracts key facts, preferences, and decisions using AI (via your own API key). Stores everything locally in your browser.

▸ **Inject** — Automatically provides relevant memories to your AI assistant with each new message, so it remembers your preferences, projects, and context.

▸ **Privacy-First** — All data stored locally on your device. Nothing is sent anywhere unless you explicitly configure sync or extraction.

### Features
• Cross-platform conversation capture (Claude, ChatGPT, Gemini, Perplexity)
• Persistent storage that survives browser restarts
• AI-powered fact extraction (Anthropic or OpenAI API key required)
• Automatic memory injection into outgoing messages
• Beautiful popup dashboard with conversation list, fact browser, and live captures
• Export/import all data as JSON
• Optional backend sync for cloud persistence
• Full options page for detailed configuration

### How It Works
1. Install the extension
2. Visit any supported AI platform — conversations are captured automatically
3. (Optional) Add your API key in Settings to enable fact extraction
4. (Optional) Enable auto-inject to give your AI context from past conversations

### Privacy
Your data stays on your device. API keys are stored locally. The extension only communicates with services you explicitly configure. No tracking, no analytics, no ads.
