🧠 MemBrain — Privacy Policy

Last updated: February 21, 2026
Summary: Your data stays on your device. We don't collect, sell, or share any personal information. The extension only communicates with services you explicitly configure.

What Data We Collect

MemBrain captures conversations from AI platforms you visit (Claude, ChatGPT, Gemini, Perplexity) while the extension is active. This includes:

  • Message text (user and assistant turns)
  • Conversation metadata (timestamps, platform, conversation IDs)
  • Extracted facts and preferences (if you configure an API key)

Where Data Is Stored

All captured data is stored locally on your device using the browser's IndexedDB storage. Your data never leaves your device unless you explicitly enable one of these optional features:

  • Backend sync: If you configure a backend URL and API key, conversations are synced to your specified server.
  • Fact extraction: If you provide an API key for Anthropic or OpenAI, conversation text is sent to that provider's API for fact extraction.

Third-Party Services

The extension may communicate with these services only if you configure them:

  • Anthropic API — for fact extraction (if you provide an Anthropic API key)
  • OpenAI API — for fact extraction (if you provide an OpenAI API key)
  • Your backend server — for conversation sync (if you configure a backend URL)

We do not send data to any other third parties. We do not use analytics, tracking, or advertising services.

API Keys

API keys you provide are stored locally in IndexedDB on your device. They are only used to authenticate with the respective service (Anthropic, OpenAI, or your backend). They are never transmitted to us or any other party.

What We Don't Do

  • We don't collect personal information
  • We don't track your browsing activity
  • We don't sell or share any data
  • We don't use cookies or analytics
  • We don't run background processes on sites other than the configured AI platforms

Data Deletion

You can delete all stored data at any time through the extension's Settings page ("Clear All Data" button). Uninstalling the extension also removes all locally stored data.

Permissions Explained

  • storage: To save conversations and settings in IndexedDB
  • tabs: To identify which AI platform tab is active
  • alarms: For periodic sync and extraction timers
  • Host permissions (AI platform URLs): To inject the conversation capture script

Open Source

This extension is open source. You can review the complete source code to verify these privacy practices.
